Colors

We've already seen some properties in CSS that take color values. Here's an example:

            p{
                color: blue;
            }
    There are several different ways to specify colors in CSS
  • Color Keyword
  • RGB
  • RGBA
  • HSL
  • HSLA
  • Hexadecimal
